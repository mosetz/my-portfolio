# My Portfolio

A clean, single-page portfolio website showcasing my technical skills, projects, and ways to get in touch.

## About

This project is a lightweight, responsive single-page portfolio built with plain HTML, CSS, and JavaScript. It highlights skills, showcases projects or work samples, and provides contact information so visitors can reach out.

## Features

- Responsive single-page layout
- Skills and technology summary
- Project or work showcase section
- Contact section with email and social links
- Easy to customize and deploy

## Tech Stack

- HTML
- CSS
- Tailwind CSS
- JavaScript


- Update content in `index.html` to add or change sections.
- Modify styles in `style.css` for colors, spacing, and layout.
- Edit `script.js` to change interactive behavior or add new features.
